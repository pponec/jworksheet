jWorkSheet

version ${VERSION}

The jWorkSheet is a desktop application intended for a tracking time of your projects,
data are presented in a spreadsheet like format. Application offers some completed HTML reports,
however you can create your own user reports by a XSL transformations.

Basic Information

  • License: license.txt file
  • Description: home page
  • Release notes: changes.txt file
  • Launch jWorkSheet from command line: java -jar jWorkSheet.jar <workingDirectory>
    where the <workingDirectory> is an optional parameter to store data files.
  • System environment: Java Runtime JRE 5.0+
  • JavaDoc: javadoc directory
  • Java Source: the sources are available include a project files for NetBeans IDE 6.1
  • Library: UJO Framework, http://ujoframework.org/
  • Icons: Mark James' icons (Creative Commons Attribution License) http://famfamfam.com
